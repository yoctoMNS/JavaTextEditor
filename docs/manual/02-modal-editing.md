# 2. モーダル編集

このエディタはVim式のモーダル編集を採用しています。モードは `ModalEditor` が管理し、キーバインドは `KeymapRegistry` によってモードごとに一元管理されています（プラグインから変更・追加可能。詳細は [プラグインシステム](07-plugins.md)）。

## モード一覧

| モード | 説明 | 入り方 |
|---|---|---|
| NORMAL | カーソル移動・コマンド発行が中心のデフォルトモード | 起動時・各モードから `Esc` |
| INSERT | 文字入力モード | NORMALで `i`/`a`/`o` など |
| COMMAND | `:` コマンド入力 | NORMALで `:` または `;` |
| VISUAL | 文字単位の範囲選択 | NORMALで `v` |
| VISUAL LINE | 行単位の範囲選択 | NORMALで `V` |
| VISUAL BLOCK | 矩形（列範囲）選択 | NORMALで `Ctrl+V` |
| SEARCH | `/` によるバッファ内検索パターン入力 | NORMALで `/` |
| FILESEARCH | `\f`/`\g` によるファイル名/grep検索パターン入力 | NORMALで `\f` または `\g` |
| TELESCOPE | telescope.vim風ファジーファインダー | NORMALで `SPC+f`/`SPC+/`/`SPC+b` |
| FILER | ディレクトリブラウザ | `:cd` 成功後に自動遷移 |

SEARCH/FILESEARCH/TELESCOPE/FILERは [検索・ナビゲーション](03-search-and-navigation.md) で詳しく扱います。

## NORMALモード

### カーソル移動

| キー | 動作 |
|---|---|
| `h` / `l` | カーソルを左/右に移動 |
| `j` / `k` | カーソルを下/上に移動 |
| `w` / `b` / `e` | 次の単語先頭 / 前の単語先頭 / 単語末尾へ移動 |
| `0` | 行の絶対先頭へ移動 |
| `^` | 行の最初の非空白文字へ移動 |
| `$` | 行末へ移動 |
| `gg` | ファイル先頭へ移動 |
| `G` | ファイル末尾へ移動 |
| `H` / `M` / `L` | 画面内の最上行/中央行/最下行へジャンプ |
| `%` | カーソル位置の括弧（`()` `[]` `{}`）に対応する括弧へジャンプ（ネスト対応）。カーソルが括弧の上にないと動作しない |
| `Ctrl+F` / `Ctrl+B` | 1画面分下/上にスクロール |
| `Ctrl+D` | 半画面分下にスクロール |
| `Ctrl+E` / `Ctrl+Y` | 1行分下/上にスクロール |
| `Space+h` | 行の最初の非空白文字へ移動（`^` 相当） |
| `Space+l` | 行末へ移動（`$` 相当） |
| `Space+k` | ファイル先頭へ移動（`gg` 相当） |
| `Space+j` | ファイル末尾へ移動（`G` 相当） |

> **`Ctrl+U` の注意**: NORMALモードでは半ページスクロールではなく「直前のバッファへ切り替え」（`buffer.prev`）に割り当てられています。`Ctrl+P` は「次のバッファへ切り替え」（`buffer.next`）です。半ページ上スクロールはVISUAL/VISUAL LINEモードでのみ `Ctrl+U` に割り当てられています（後述）。バッファ切り替えの詳細は [検索・ナビゲーション](03-search-and-navigation.md) を参照してください。

### 編集

| キー | 動作 |
|---|---|
| `i` | INSERTモードへ（カーソル前に挿入） |
| `a` | INSERTモードへ（カーソル後に挿入） |
| `o` | 現在行の下に新しい行を開いてINSERTモードへ |
| `u` | 直前の編集を取り消す（アンドゥ） |
| `Ctrl+R` | 取り消した編集をやり直す（リドゥ） |
| `yy` | 現在行をヤンク |
| `dd` | 現在行を削除してヤンク |
| `x` | カーソル位置の1文字を削除 |
| `p` | ヤンクレジスタの内容をカーソルの後ろ（文字）または現在行の下（行）に貼り付け |
| `P` | ヤンクレジスタの内容をカーソルの前（文字）または現在行の上（行）に貼り付け |
| `Alt+J` | 現在行と次行を入れ替え |
| `Alt+K` | 現在行と前行を入れ替え |

### モード遷移・その他

| キー | 動作 |
|---|---|
| `v` | VISUALモードへ |
| `V` | VISUAL LINEモードへ |
| `Ctrl+V` | VISUAL BLOCKモードへ |
| `gv` | 直前のVISUAL系選択範囲を復元して再度選択状態に戻る |
| `:` / `;` | COMMANDモードへ |
| `K` | 定義・ドキュメント参照（[Java開発支援](04-java-tooling.md)参照） |
| `Shift+J` | 直前の `K` ジャンプ元へ戻る（[Java開発支援](04-java-tooling.md)参照） |
| `gr` / `gR` | 参照検索（[検索・ナビゲーション](03-search-and-navigation.md)参照） |

その他、Getter/Setter生成・import整理・診断ジャンプ・ペイン操作・検索系キーは各章を参照してください（[編集支援機能](06-editing-features.md)、[Java開発支援](04-java-tooling.md)、[検索・ナビゲーション](03-search-and-navigation.md)）。

## INSERTモード

| キー | 動作 |
|---|---|
| 通常文字 | カーソル位置に文字を挿入 |
| `Backspace` | カーソル直前の文字を削除（行頭では前行と結合） |
| `Enter` | 改行を挿入（前行のインデントを引き継ぎ、`{` の直後ならさらに1段インデント） |
| `Escape` | NORMALモードへ復帰 |
| `Ctrl+F` / `Ctrl+B` | カーソルを右/左に移動（Emacs式） |
| `Ctrl+N` / `Ctrl+P` | カーソルを下/上に移動（Emacs式） |
| `Ctrl+A` | 行の最初の非空白文字へ移動（Emacs式） |
| `Ctrl+E` | 行末へ移動（Emacs式） |
| `Alt+F` / `Alt+B` | 次/前の単語先頭へ移動（Emacs式） |
| `Ctrl+Home` / `Ctrl+End` | ファイル先頭/末尾へ移動（Emacs式） |
| `Tab` | 4スペースを挿入。閉じ括弧（`)` `]` `}` `"` `'` `>`）の直前ではスペース挿入をせず外側へスキップ |
| `}` | 行がインデントのみの場合は1段分自動デデントしてから `}` を挿入 |
| `Ctrl+D` | カーソル位置の1文字を削除（Emacsの `delete-char` 相当） |
| `Ctrl+K` | カーソルから行末まで削除（Emacsの `kill-line` 相当） |
| `Ctrl+W` | カーソル直前の1単語を削除（Vim互換、行頭をまたがない） |
| `Ctrl+]` / `Ctrl+[` | NORMALモードへ戻りつつ即座にファイル保存 |
| `Ctrl+Space` | コード補完ポップアップ起動（[コード補完](05-completion.md)） |
| `Alt+/` | 単語補完ポップアップ起動（[コード補完](05-completion.md)） |
| `Ctrl+Shift+O` | 未使用importを一括削除（[Java開発支援](04-java-tooling.md)） |

## COMMANDモード

`:` または `;` でCOMMANDモードに入り、画面下部にコマンド入力欄が表示されます。文字入力・`Backspace`・`Tab`（`:cd`/`:e` のパス補完）はキーマップを経由せず直接処理されます。

| キー | 動作 |
|---|---|
| 通常文字 | 入力済みコマンドに文字を追加 |
| `Backspace` | 入力済みコマンドを1文字削除 |
| `Tab` | `cd`/`e` コマンドのパス名を補完（[検索・ナビゲーション](03-search-and-navigation.md)参照） |
| `Enter` | コマンドを実行してNORMALモードへ戻る |
| `Escape` | COMMANDモードを中断してNORMALモードへ戻る |

コマンド一覧は各章に分散しています（`:w`/`:e`/`:q` 等の基本操作は本ページ末尾、`:grep`/`:rename`/`:cd` は [検索・ナビゲーション](03-search-and-navigation.md)、`:main`/`:oi` は [Java開発支援](04-java-tooling.md)、`:tutor` は [チュートリアル](08-tutorial.md)）。網羅的な一覧は [キーバインド早見表](10-keybindings-reference.md) を参照してください。

### 基本コマンド

| コマンド | 動作 |
|---|---|
| `:w` | 現在のファイルパスへ上書き保存 |
| `:w <path>` | 指定したパスへ保存（以降そのパスが「現在のファイル」になる）。`s/pattern/replacement/` によるファイル名の正規表現置換にも対応 |
| `:e` / `:enew` | 新規の空バッファを開く |
| `:e <path>` | 指定したファイルを開く（バッファを差し替え・カーソルをリセット） |
| `:sp` / `:split` | 垂直分割（ペインを左右に分割） |
| `:vs` / `:vsplit` / `:vsp` | 水平分割（ペインを上下に分割） |
| `:q` | 現在のペインを閉じる（最後のペインならエディタを終了） |
| `:wq` | 保存してから `:q` |

### 置換コマンド（`:s`）

Vim式の `:s`（substitute）コマンドで、正規表現による文字列置換ができます。

| コマンド | 対象範囲 |
|---|---|
| `:s/pattern/replacement/` | カーソル行のみ |
| `:%s/pattern/replacement/` | バッファ全行 |
| `:N,Ms/pattern/replacement/`（1始まり） | N行目〜M行目 |
| `:'<,'>s/pattern/replacement/` | 直前のVISUAL選択範囲の行 |

- 区切り文字は `/` 以外の任意の1文字も使えます（例: `:s#foo#bar#`）
- フラグ（末尾に付与）: `g` で行内の全マッチを置換（省略時は各行の最初のマッチのみ）、`i` で大文字小文字を無視
- 置換文字列側で `&` はマッチ全体、`\1`〜`\9` は後方参照として使えます（`\&` でリテラルの `&`）
- パターンを空にすると、直前の `/` 検索（[検索・ナビゲーション](03-search-and-navigation.md)）で使ったパターンを再利用します
- 置換後はマッチ件数と対象行数がステータスバーに `N substitutions on M lines` の形式で表示され、カーソルは最後に変更された行へ移動します。マッチが1件もなければ `E: pattern not found: <pattern>` を表示します
- VISUAL/VISUAL LINE/VISUAL BLOCKモード中に `:` を押すと、選択範囲を表す `'<,'>` が自動入力された状態でCOMMANDモードに入ります（後述）

## VISUALモード（文字単位選択）

| キー | 動作 |
|---|---|
| `h` / `l` / `j` / `k` | 選択範囲を拡張 |
| `w` / `b` / `e` | 単語単位で選択範囲を拡張 |
| `0` / `^` / `$` | 行頭/最初の非空白文字/行末まで選択範囲を拡張 |
| `G` | ファイル末尾まで選択範囲を拡張 |
| `%` | 対応する括弧まで選択範囲を拡張 |
| `Ctrl+F` / `Ctrl+B` | 1画面分下/上にスクロール |
| `Ctrl+D` / `Ctrl+U` | 半画面分下/上にスクロール |
| `>` / `<` | 選択行をインデント/デデント（`shiftwidth` 基準、数字プレフィックスで段数指定可） |
| `y` | 選択範囲をヤンク（文字単位）してNORMALモードへ |
| `d` | 選択範囲を削除してヤンク（文字単位）し、NORMALモードへ |
| `:` | 選択範囲を表す `'<,'>` を自動入力した状態でCOMMANDモードへ（`:s` 置換コマンド用。上記参照） |
| `v` / `Escape` | VISUALモードを解除してNORMALモードへ |

|---|---|
| `h` / `l` / `j` / `k` | 選択行範囲を拡張 |
| `G` | ファイル末尾まで選択行範囲を拡張 |
| `Ctrl+F` / `Ctrl+B` | 1画面分下/上にスクロール |
| `Ctrl+D` / `Ctrl+U` | 半画面分下/上にスクロール |
| `>` / `<` | 選択行をインデント/デデント |
| `y` | 選択行をヤンク（行単位）してNORMALモードへ |
| `d` | 選択行を削除してヤンク（行単位）し、NORMALモードへ |
| `:` | 選択範囲を表す `'<,'>` を自動入力した状態でCOMMANDモードへ（`:s` 置換コマンド用） |
| `V` / `Escape` | VISUAL LINEモードを解除してNORMALモードへ |

## VISUAL BLOCKモード（矩形選択、`Ctrl+V`）

Vimの矩形選択（`Ctrl+V`）相当の機能です。複数行の同じ列範囲だけを選択・編集できます。

| キー | 動作 |
|---|---|
| `h` / `j` / `k` / `l` | 矩形の範囲を拡張 |
| `%` | 対応する括弧まで矩形の列範囲を拡張 |
| `y` | 矩形範囲をヤンク（ブロック型として記録）し、カーソルは矩形左上へ、NORMALモードへ戻る |
| `d` | 矩形範囲を各行から削除（ヤンクも兼ねる）し、NORMALモードへ戻る |
| `I` | 矩形の**左端**の列でINSERTモードへ。入力内容は選択していた全行に同時反映される（複数行同時入力） |
| `A` | 矩形の**右端**の1つ右の列でINSERTモードへ。同様に全行へ同時反映される |
| `c` | 矩形範囲を削除したうえで `I` と同様の位置でINSERTモードへ（矩形の内容を置き換える） |
| `r` | 次に押した1文字で、矩形が覆う全位置の文字を一括置換 |
| `>` / `<` | 選択行をインデント/デデント |
| `:` | 選択範囲を表す `'<,'>` を自動入力した状態でCOMMANDモードへ（`:s` 置換コマンド用。矩形の列範囲ではなく選択していた行全体が対象になります） |
| `Ctrl+V` / `Escape` | VISUAL BLOCKモードを解除してNORMALモードへ |

**矩形貼り付け（NORMALモードでの `p`/`P`）**: ヤンクレジスタがブロック型の場合、`p`/`P` は行ごとに分割された内容を列位置を揃えて貼り付けます。貼り付け先の行数がバッファの行数を超える場合は自動的に新しい行を追記し、貼り付け列に満たない短い行はスペースでパディングしてから挿入します。`p` はカーソル列の右隣から、`P` はカーソル列から貼り付けます。

**`I`/`A`/`c` の複数行同時入力に関する制約**: INSERT中に `Enter`（改行）を押すと、複数行への同時反映は中断されます（1行分の入力のみがその行に反映され、以降の行には反映されません）。単一行のテキスト追加・削除のみが全行に複製される設計です。
