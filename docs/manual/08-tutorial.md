# 8. チュートリアル

`:tutor`（または `:tutorial`）コマンドで、vimtutor形式の対話型チュートリアルを開けます。

```
:tutor
```

## 特徴

- 説明を読みながら実際にそのバッファを編集して学ぶ形式です。移動・削除・挿入・ヤンク/ペースト・VISUALモード・検索・コマンドモードといったVim由来の操作から、このエディタ独自のEmacs式移動・Spaceリーダーキー・Java編集支援・プロジェクト機能まで一通り体験できます
- **自動採点はしません**。vimtutorと同じ設計判断で、キー入力の正誤判定や進捗トラッキングの仕組みは持ちません（学習目的のシンプルさを優先するため、専用モードや状態機械を追加していません）
- 本文は `dev.javatexteditor.tutorial.Tutorial.CONTENT` にJavaのテキストブロックとして埋め込まれています（ビルドスクリプトが `.java` ファイルしかコンパイルしないため、外部の `.txt` ファイルは同梱していません）

## 操作

| キー/コマンド | 動作 |
|---|---|
| `:tutor` / `:tutorial` | チュートリアルを開く（現在のバッファは履歴に退避される） |
| `Ctrl+U` | チュートリアルを開く前のバッファに戻る（既存のバッファ履歴機構を利用） |
| `:q` | エディタを終了（チュートリアルバッファも通常のバッファとして扱われるため、保存しようとすると「保存先がない」エラーになります） |

チュートリアルは `:enew`（`newBuffer()`）や `:grep`（`executeGrep()`）と同じ「疑似バッファ」パターンで実装されており、`pushBuffer()` で現在のバッファを履歴に積んでから新規バッファに差し替えます。
